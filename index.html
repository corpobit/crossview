Crossview Helm Chart Repository

This repository hosts Helm charts for Crossview - A Kubernetes resource visualization and management platform.

Add the Repository

helm repo add crossview https://corpobit.github.io/crossview
helm repo update

Install Crossview

helm install crossview crossview/crossview \
--namespace crossview \
--create-namespace

For more information, visit the project repository.

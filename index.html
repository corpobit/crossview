Crossview Helm Charts

Official Helm repository for Crossview – Crossplane visualization platform

Install from OCI Registry (Recommended)

helm install crossview oci://docker.io/corpobit/crossview-chart \
--namespace crossview \
--create-namespace

Add Helm Repository

helm repo add crossview https://corpobit.github.io/crossview
helm repo update

Install from Repository

helm install crossview crossview/crossview \
--namespace crossview \
  --create-namespace

Upgrade

helm upgrade crossview crossview/crossview \
  --namespace crossview
